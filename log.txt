você sabe o que é lgpd?

Sim, eu sei o que é LGPD. A LGPD (Lei Geral de Proteção de Dados) é uma lei brasileira que estabelece um conjunto de regras para o tratamento de dados pessoais. Ela cria novos conceitos jurídicos, estabelece as condições nas quais os dados pessoais podem ser tratados, define um conjunto de direitos para os titulares dos dados, gera obrigações específicas para os controladores dos dados e cria uma série de procedimentos e normas para que haja maior cuidado com o tratamento de dados pessoais e compartilhamento com terceiros.
